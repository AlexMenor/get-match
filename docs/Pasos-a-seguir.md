# Pasos a seguir

En este documento se recogerán los pasos a seguir a lo largo del desarrollo del proyecto de la asignatura:

- Se crearán todas las clases/entidades necesarias para la correcta representación.
- Elegir tipo de persistencia a usar para el almacenamiento de los datos.
- Elección de bibliotecas y otras herramientas de forma específica.
- Implementar las operaciones de consulta.
- Búsqueda de fuente de datos que me permita estar lo máximo posible al día.
- Creación de sistema de log.
- Proporcionar al cliente la posibilidad de hacer las consultas.
